### Старт

Клонировать репозиторий
```bash
$ git clone https://github.com/bulgakov-hub/Blog.git
$ cd Blog
```
Создать образ
```bash
$ docker-compose build
```
Запустить контейнер
```bash
$ docker-compose up
```
Перейти по адресу
```
http://localhost:8000
```
Войти от Администратора
```
username: admin
password: admin
```
